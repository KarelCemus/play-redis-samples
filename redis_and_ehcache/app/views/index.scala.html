Redis cache with EhCache example!

This application shows the use of Redis simultaneously with EhCache using play-redis.

All messages are cached for 5 seconds. All messages are cached under the same key in different storages.

First, there is a EhCache instance bound to unqualified API with the current message: @ehcache

Second, there is a Redis cache boud to @@NamedCache("redis"). The current message is @redis

Right now, it's @now.
